# pedroarantes.com
   WordPress theme
   
## Tech stack
* WordPress 
* Gulp.js
* Stylus css
* jQuery
* Slicky slider
* WP plugin ACF ( Advanced custom fields ) 

## Start local project
On terminal:

     $ npm install
     $ gulp
